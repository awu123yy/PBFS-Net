# PBFS-Net

This repository provides an anonymized review version of the code for the manuscript:

**Plan-Geometry-Aware Three-Dimensional Radiotherapy Dose Distribution Prediction Using a Pseudo Beam Prior-Guided Network**

## Current status

The current version includes the core network implementation of PBFS-Net for peer review. The complete public release will be updated upon acceptance, including the full training pipeline, preprocessing scripts, pseudo beam prior construction procedure, inference code, evaluation scripts, configuration files, and detailed usage instructions.

## Code availability

This repository is provided for double-anonymous peer review. The final public repository will be updated and maintained after acceptance.
